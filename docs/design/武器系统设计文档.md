# CYBER BLADE · 武器系统设计文档

> 参考《土豆兄弟》(Brotato) 武器/标签/羁绊体系，结合项目现有代码框架
> 版本：v1.0 — 策划案

---

## 一、设计目标

1. **复制 Brotato 的武器标签+羁绊体系**：同标签武器数量触发属性加成，引导玩家凑流派
2. **武器池扩充至 20 种以上**：覆盖 6+ 标签，每个标签 3~6 把武器
3. **道具池扩充至 20 种以上**：涵盖 Brotato 经典道具类型（正负效果组合）
4. **清晰的武器稀有度/等级体系**：1~4 级品质，同名武器可合并升级
5. **保持现有代码架构**：武器定义在 `shop.js`，行为在 `player.js`，渲染在 `renderer.js`

### 2.2 本项目的标签定义（精简版）

```
现有标签：
  rapid  ⚡ 速射  → 羁绊：攻速/弹速流派
  heavy  💥 重火  → 羁绊：伤害/穿透流派

  medic  💊 医疗  → 羁绊：生命恢复/吸血流派
```

## 七、代码参考

### 7.1 武器定义模板（shop.js）

```js
{
    id: 'axe',                  // 唯一ID
    name: '能量斧',             // 显示名
    desc: '扇形横扫 / +20%暴击伤害',  // 描述
    icon: '🪓',                 // 回退图标
    tag: 'melee',               // 标签
    slots: 2,                   // 占用槽位
    cost: 12,                   // 价格
    mods: { damageMult: 0.2, critMultiplierAdd: 0.5 },  // 属性修正
    behavior: 'melee',          // 行为类型
    bulletCount: 1,
    bulletSpeed: 0,
    damageMult: 2.0,
    attackSpeedMult: 1.1,
    spread: 0,
    pierce: 0,
    meleeRange: 100,
    unlock: { type: 'default' } // 解锁条件
}
```

### 7.2 新增 weaponRarity（武器品质分级）

4 级品质，对应不同基础数值：

| 品质 | 颜色 | 基础伤害倍率 | 出现波次 | 基础价格倍率 |
|------|------|-------------|---------|-------------|
| 普通 (T1) | ⬛ 黑色 | 1.0x | 1 | 1.0x |
| 优秀 (T2) | 🟦 蓝色 | 1.3x | 3 | 1.8x |
| 稀有 (T3) | 🟪 紫色 | 1.6x | 6 | 2.8x |
| 传说 (T4) | 🟥 红色 | 2.0x | 10 | 4.0x |

2 个同名同品级武器可合成更高品级（+1 级）。

### 7.3 独特道具逻辑（shop.js）

```js
{
    id: 'energy_shield',
    name: '能量盾',
    desc: '每 8 秒抵挡 1 次伤害',
    cost: 10,
    icon: '🛡️',
    unique: true,  // ← 标记为独特道具
    apply: (p) => {
        p.energyShieldCooldown = 8;
        p.energyShieldTimer = 0;
        // 在 player.takeDamage() 中添加护盾检测
    }
}
```

购买独特道具后，从 `allItems[]` 临时移除防止再次刷新。
