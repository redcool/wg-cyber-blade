> 参考《土豆兄弟》(Brotato) 武器/标签/羁绊/角色体系

---

## 一、设计目标

1. **扩充武器池至 48 种**：10 近战 + 10 枪械 + 10 弓箭 + 13 元素 + 5 医疗 + 3 骑枪
2. **角色扩展至 10 个**：4 个默认解锁，6 个条件解锁
3. **角色武器适配系统**：不同角色只能选择适配类型的武器
4. **关卡结束清理机制**：倒计时结束后怪/子弹/金币消失
5. **医药箱系统**：可击破的医疗箱，掉落的药品拾取回血

### 3.3 角色武器适配系统

每个角色有一个 `weaponAffinities` 数组，表示该角色可以购买的武器类型。

- 商店中只显示该角色适配的武器
- 不适配的武器不出现在商店中
- 默认角色（游侠）适配所有类型

```js
weaponAffinities: ['melee', 'gun', 'bow', 'magic', 'medic'],  // 默认全适配
weaponAffinities: ['melee'],  // 剑客只能买近战
weaponAffinities: ['gun'],    // 枪手只能买枪械
weaponAffinities: ['magic'],  // 火焰法师只能买元素
weaponAffinities: ['bow'],    // 弓箭游侠只能买弓箭
```

## 四、倒计时结束清理机制

当关卡倒计时结束时（`WaveSystem.waveTimer >= WaveSystem.levelDuration`）：

1. 所有存活敌人立即死亡（不掉材料）
2. 所有子弹消失
3. 地面上未拾取的金币消失
4. 触发关卡过渡动画

```js
// 在 WaveSystem.update 中，当 waveTimer >= levelDuration 时：
_cleanupWave() {
    // 敌人消失（不掉材料）
    for (const e of EnemySystem.enemies) {
        if (e.alive) {
            e.alive = false;
            ParticleSystem.emit(e.x, e.y, 6, { color: e.color, speed: 50, life: 0.3, size: 4 });
        }
    }
    // 子弹清除
    BulletSystem.clear();
    // 金币清除
    GameWorld.materials = [];
}
```

## 五、医药箱系统

### 5.1 生成规则

- 每关生成 2~3 个医药箱
- 医药箱出现在地图随机位置
- 医药箱不移动，可被攻击破坏
- 医药箱有独立的 HP（30~50）

### 5.2 医疗包掉落

- 击破医药箱掉落 1~2 个医疗包
- 医疗包在地面停留 8 秒后消失
- 拾取医疗包立即恢复 15~25 HP

### 5.3 代码结构

```js
// 新文件或添加到 enemy.js
const MedkitSystem = {
    crates: [],       // 场景中的医药箱
    pickups: [],      // 掉落的医疗包
    
    spawnCrates(count) { ... },
    update(dt, player) { ... },
    takeDamage(crate, damage) { ... },
    _spawnPickups(crate) { ... },
    _pickupHealth(healthPack, player) { ... },
}
```
