## 一、射击大全 3 大类 → 1 级 class 映射

| 射击大全分类 | 1 级 class (本项目) | 2 级 class_2 词汇 | 游戏机制特征 |
|--------------|---------------------|------------------|--------------|
| **壹 弓类** (弹性势能) | `Bow` | longbow, recurve, composite | 长距精准,可蓄力 |
| **贰 弩类** (机械弹射) | `Crossbow` | handcrossbow, heavycrossbow, repeating | 装填慢,穿透强 |
| **叁 现代枪械** (火药) | `Gun` | pistol, revolver, smg, rifle, sniper, shotgun, lmg, hmg | 高弹速,弹道直 |

> **关键拆分**: 在本项目 6 标签中,`bow` 单一标签对应了**弓 + 弩 + 早期枪械**三类机制差异巨大的武器族,不利于玩家理解。拆为 `Bow` / `Crossbow` / `Gun` 3 个 1 级 class 是必要的。

## 二、射击武器核心机制 (5 维度)

### 2.1 弹道类型 (bulletBehavior)

| behavior | 中文 | 适用 class | 现有武器 |
|----------|------|-----------|----------|
| `bullet` | 标准子弹 | Gun.pistol/rifle, Bow, Crossbow | pistol, bow, crossbow |
| `spread` | 散射 | Gun.shotgun, Crossbow.repeating | shotgun, shotgun_double, triple_shot |
| `burst` | 连射 | Gun.rifle/smg | rifle, smg |
| `auto` | 全自动 | Gun.smg/lmg/hmg | smg, gatling, minigun |
| `pierce` | 穿透 | Bow.longbow, Gun.sniper, Crossbow.heavycrossbow | longbow, sniper, piercing_shot |
| `homing` | 追踪 | Bow (魔法), Crossbow (魔法), Elemental | homing_bow, magic_orb |
| `explode` | 爆炸 | Explosive, Gun.rocket | explosive_arrow |

### 2.2 弹速 (bulletSpeed)

| 等级 | 值 (px/s) | 适用 class | 备注 |
|------|----------|-----------|------|
| 极慢 | 300~400 | homing_bow, shotgun | 慢弹易躲,适合追踪/散射 |
| 慢 | 400~500 | shotgun, frost_arrow, explosive_arrow | |
| 中 | 500~700 | pistol, bow, crossbow | 标准 |
| 快 | 700~900 | rifle, smg, recurve | |
| 极快 | 1000~1200 | sniper, minigun | 几乎瞬到 |

### 2.3 弹数 (bulletCount)

| 弹数 | 适用 | 备注 |
|------|------|------|
| 1 (单发) | pistol, rifle, sniper, longbow | 标准 |
| 2~3 | smg, gatling, triple_shot | 散射 |
| 4~6 | shotgun, gatling, shotgun_double | 扇形 |
| 8+ | shotgun_double | 极端散射,低单发伤害 |
| 多发 (3 发连射) | rifle (3-burst) | 一次扣动多发 |

### 2.4 穿透 (pierce)

| 等级 | 值 | 适用 | 备注 |
|------|-----|------|------|
| 无 | 0 | shotgun, smg, homing_bow | 单体 |
| 弱 | 1~2 | bow, pistol, rifle, sniper | 偶有穿透 |
| 中 | 3~4 | longbow, crossbow | 稳定 |
| 强 | 5~6 | piercing_shot, sniper (T2 升满) | 清场 |

### 2.5 特殊效果

| 效果 | 适用 | 现有武器 |
|------|------|----------|
| **暴击加成** | bow (高暴击族) | bow, longbow, recurve |
| **爆炸** | explosive, 某些 gun | explosive_arrow, magnum (穿透+爆) |
| **减速** | frost 系列 | frost_arrow, frost_staff |
| **中毒** | poison 系列 | poison_arrow, poison_staff |
| **追踪** | homing, 魔法 | homing_bow, magic_orb |
| **吸血** | 生命偷取 | (一般走 lifesteal 词缀) |

## 三、3 大类的差异化体验

| 维度 | Bow (弓) | Crossbow (弩) | Gun (枪) |
|------|----------|---------------|----------|
| **典型射速** | 1.0~1.5/s | 0.5~0.8/s | 2.0~10/s (auto) |
| **典型伤害** | 中 (×1.2~1.5) | 高 (×1.8~2.5) | 低~中 (×0.6~1.2) |
| **穿透** | 弱~中 | 中~强 | 弱~中 (除 sniper) |
| **特殊优势** | 高暴击,暴伤加成 | 高穿透,单发强力 | 高射速,持续输出 |
| **Build 路线** | 暴击流,精准流 | 穿透流,清场流 | 射速流,DPS 流 |
| **典型角色** | 弓箭游侠 (archer) | (待分配,目前无专属) | 枪手 (gunslinger) |
| **现实映射** | 后羿/罗宾汉 | 诸葛连弩/欧洲板弩 | 西部牛仔/特种兵 |

> **新增建议**: 弩 (Crossbow) 缺少专属角色,建议在 v1.1 角色扩展时增加 **弩手 (crossbowman)** 角色,弥补"中距离高穿透"Build 路线空白。

## 五、设计原则 (KISS)

1. **弓/弩/枪 不能合并**: 三者机制差异巨大 (蓄力/装填/换弹),合并会让玩家困惑
2. **shotgun 散射不算独立 class**: 它是 Gun 下的 shotgun 2 级,因散射 vs 连射在玩家看来都属于"枪"
3. **弹速 1000+ 才能叫"快"**: 400~500 算慢/标准,避免游戏内"快"标准不一致
4. **穿透 vs 弹数 平衡**: 穿透多武器弹数少 (sniper), 弹数多武器穿透少 (shotgun), 中间型 (rifle) 取平衡

## 六、角色 (v1.1 新增弩手)

### 6.1 弩手 crossbowman ⭐ v1.1 正式角色

> **设计意图**: 弥补"中距离高穿透"Build 路线空白。弓箭游侠 偏 bow 2级 (longbow/recurve),弩手 偏 crossbow 2级 (handcrossbow/heavycrossbow/repeating)。

```js
{
  id: 'crossbowman',
  name: '弩手',
  icon: '🎯',
  tags: ['bow'],                                  // 兼容旧,匹配 weapon.tag = 'bow'
  preferredClasses: ['Crossbow', 'Bow'],           // 1 级偏好: 弩 + 弓
  preferredClasses_2: ['handcrossbow', 'heavycrossbow', 'repeating', 'longbow'],  // 2 级偏好: 3 弩 + 1 弓 fallback
  weaponAffinities: ['bow'],                       // 商店过滤: bow (跟 archer 共享)
  maxWeapons: 6,                                   // 槽位简化: 1 武器 = 1 槽
  baseStats: { hp: 100, damage: 18, attackSpeed: 0.9, range: 380, critChance: 8%, critDamage: 2.5 },
  unlockCondition: '累计击杀≥60',                  // 易于解锁 (assassin ≥100)
}
```

> **与弓箭游侠的区分**:
> - 同 `tags: ['bow']` + `weaponAffinities: ['bow']` → 共享商店武器池
> - 1 级偏好相同 (`Bow, Crossbow`),但 2 级偏好侧重不同:
>   - 弓箭游侠 2级: longbow / recurve / handcrossbow / heavycrossbow → bow 偏多
>   - 弩手 2级: handcrossbow / heavycrossbow / repeating / longbow → crossbow 偏多
> - **结果**: 弩手拿 crossbow 类 1.0,弓箭游侠拿 bow 类 1.0, 交叉各 0.85 (1 级)
>
> 详见 [角色-class偏好 §二.5](./角色-class偏好.md)

### 6.2 未来扩展 (v1.2+)

- **弹道可视化升级**:
  - **追踪轨迹**: homing 弹道显示预测线
  - **穿透残留**: pierce 弹在穿透敌人时显示穿透粒子
  - **散射扩散**: spread 弹在飞行过程中扩散可视化
- **新武器族** (基于射击大全):
  - `compound_crossbow` 复合弩 (Crossbow.composite 类, 新增)
  - `railgun` 电磁轨道炮 (Gun.sniper 类, 科技元素)
  - `flamethrower` 火焰喷射器 (已在 magic 类别, 可挪到 gun)
