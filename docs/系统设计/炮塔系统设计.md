# 炮塔系统设计

> 创建: 2026-06-08 | 状态: 设计阶段, 待实施

## 目标

Brotato 风格: 击破容器(树木) → 建造炮塔(需要特定道具). 炮塔自动攻击敌人, 不可被敌人摧毁, 伤害有限.

## 玩家需求

- 道具 "炮塔建造器" 激活后, 击破容器时掉落 1 级炮塔
- 炮塔: 固定实体, 自动瞄准最近敌人, 发射子弹, 有持续伤害但有限
- 敌人不可摧毁炮塔
- 炮塔只有 1 级 (简单, 不升级)

## 设计

### 1. 道具: 炮塔建造器 (Turret Builder)

```json
{
  "id": "turret_builder",
  "name": "炮塔建造器",
  "desc": "击破医疗箱时, 在位置建造 1 级炮塔",
  "rarity": "rare",
  "icon": "🏗️",
  "tags": ["engineering"],
  "statMods": {},
  "unique": true
}
```

### 2. 炮塔实体 (Turret)

```ts
interface Turret {
  x: number;           // 位置
  y: number;
  radius: number;      // 碰撞半径 (12px)
  range: number;       // 攻击范围 (200px)
  damage: number;      // 伤害 (5 基础)
  attackCooldown: number;  // 攻击间隔 (1.5s)
  attackTimer: number;     // 当前冷却
  alive: boolean;
  visualRotation: number;  // 朝向角度
}
```

### 3. 交互流程

```
容器击破 (ContainerSystem.takeDamage)
  ↓ player 有 turret_builder 道具?
  ↓ YES
  ↓
TurretSystem.spawn(container.x, container.y)
  ↓
Turret 加入 turrets[] 数组
  ↓ 每帧
TurretSystem.update(dt, enemies)
  → 找最近敌人 (距离 < range)
  → 冷却计时
  → 发射子弹 (BulletSystem.createFriendly)
```

### 4. 子弹系统扩展

需要 `BulletSystem.createFriendly` → 创建友方子弹 (isPlayer=false, 命中敌人扣血, 不命中玩家):
- 复用现有 BulletSystem.create 逻辑
- 加 `isFriendly: true` 标记
- 碰撞检测时: 友方子弹跳过玩家碰撞, 命中敌人

### 5. 渲染

`Renderer.drawTurret(turret)`:
- 圆形底座 (灰色) + 炮管 (朝向最近敌人)
- 简单矢量: ctx.arc + ctx.fillRect

### 6. 文件结构

| 文件 | 内容 |
|---|---|
| `src/engine/turret.js` | TurretSystem (新建) |
| `src/engine/bullet.js` | +createFriendly, 碰撞扩展 |
| `src/engine/renderer.js` | +drawTurret |
| `src/engine/container.js` | ContainerSystem.takeDamage 钩子: 检查 turret_builder |
| `src/cyberblade/main.js` | _updatePlaying: +TurretSystem.update |
| `src/engine/engine.js` | 渲染循环: +TurretSystem.turrets |
| `src/data/items.json` | +turret_builder |
| `index.html` | +turret.js |

### 7. 注意事项

- 炮塔不计入敌人 kill 统计
- 炮塔在波次结束后不清除 (保留到下一波)
- 炮塔数量上限: 建议 5 个 (防止性能问题)
- 炮塔子弹速度: 300px/s (比玩家子弹慢)
- 炮塔伤害不享受玩家属性加成 (独立基础值)
