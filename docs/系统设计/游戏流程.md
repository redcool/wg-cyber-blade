## 一、关卡循环

```
┌─────────────────────────────────────────────────────────────────────┐
│ 第 N 关开始 (WaveSystem.startNextLevel)                            │
│   ↓                                                                  │
│ 60 秒倒计时 + 持续刷怪 (Brotato 风格, 不允许"杀光提前结束")          │
│   ├─ Boss 波 (currentLevel % 5 === 0) → 4s 后生成 Boss              │
│   ├─ 倒计时归零 → endWave() → 清场 → waveTransitioning = true       │
│   ↓                                                                  │
│ 玩家拾取地上的所有宝箱 (精英 / Boss 掉落)                            │
│   ├─ 选完所有宝箱奖励 → onAllChestsOpened 回调                       │
│   ↓                                                                  │
│ 流转判定:                                                            │
│   ├─ 通关判定 (见 §二)                                               │
│   ├─ levelUpPending (玩家升级) → LevelUp overlay → 再回到下面        │
│   └─ 默认 → Shop overlay                                             │
│   ↓                                                                  │
│ 玩家在商店买/卖/合 → 点"进入下一关" → closeShop() → 第 N+1 关        │
└─────────────────────────────────────────────────────────────────────┘
```

## 二、通关条件 (MAX_LEVEL = 20)

**普通模式**:
- 第 1~19 关: 标准关卡循环 (5/10/15/20 都是 Boss 波)
- 第 20 关 Boss 被打死 → 掉落传奇宝箱 (legendary)
- 玩家拾取宝箱后, **LootSystem.onAllChestsOpened** 回调检查:
  - `WaveSystem.isFinalLevel()` 为 true (currentLevel >= MAX_LEVEL=20)
  - `GameEngine.endlessMode` 为 false
  - → `state = 'victory'`, `UISystem.showVictory()` 显示结算界面

**结算界面 (`#victoryOverlay`)**:
- 标题: 🏆 系统通关! (绿色胜利色 `#00ff88`)
- 信息: "击败第 20 关 Boss · 角色完成通关"
- 统计: 角色 / 击杀 / 材料 / 本局武器 / 新解锁 (复用 gameOverOverlay 字段)
- 双按钮: "🔄 再来一局" (→ 角色选择) + "🏠 返回主菜单"

**关键代码**:
- `WaveSystem.MAX_LEVEL = 20` ([src/engine/wave.js:534](src/engine/wave.js))
- `WaveSystem.isFinalLevel()` ([src/engine/wave.js:498](src/engine/wave.js))
- `GameEngine.endlessMode` 标志 ([src/engine/engine.js:20](src/engine/engine.js))
- `LootSystem.onAllChestsOpened` 结算分支 ([src/cyberblade/main.js:88](src/cyberblade/main.js))
- `UISystem.showVictory()` ([src/cyberblade/ui.js:927](src/cyberblade/ui.js))

## 三、无尽模式 (Endless Mode)

**入口**: 第 19 关商店 (currentLevel === MAX_LEVEL - 1 = 19)
- 商店按钮区出现额外的"♾️ 无尽模式"按钮 (淡紫色高亮)
- "进入下一关" 按钮 = 普通模式 (通关路径)
- "无尽模式" 按钮 = 跳过 20 关的通关判定, 转入无尽

**无尽模式下**:
- `GameEngine.endlessMode = true`
- `closeShop(endless=true)` 接受参数
- 第 20 关 Boss 死亡: `BossSystem.destroy` 检测 `endlessMode && currentLevel >= MAX_LEVEL` → 掉落 **高级宝箱** (advanced, 不是传奇)
- 玩家拾取高级宝箱 → onAllChestsOpened → `isFinalLevel && !endlessMode` 为 false → 走 levelup/shop 正常流程
- 第 21+ 关继续, 无上限, Boss 每 5 关出现, 难度持续升高

**重置**:
- `GameEngine.startGame` 内 `this.endlessMode = false` 强制重置
- 不与存档互通 (单局标志)

**高级宝箱 (advanced)**:
- 稀有度权重: common=0 / rare=5 / epic=25 / **legendary=70** (70% 必出传奇)
- 选项数: 4 个 (高于传奇的 3 个)
- 金币区间: 100~250 (高于传奇的 50~100)
- 武器初始 quality: T4 (顶级)
- 颜色: magenta `#ff00ff` (区分于传奇橙)
- 落点: 无尽模式下第 20 关 Boss 死亡 (其他场景不会触发)

## 四、商店卖武器安全

**问题**: 旧实现允许玩家把最后 1 把武器也卖掉, 导致 `player.weapons.length === 0` 的非法状态。

**修复** (v1.2):
- `ShopSystem.sellWeapon(slotIdx)` 开头加检查: `if (player.weapons.length <= 1) return false;`
- `UISystem._showWeaponDetailModal`: `canSell = ownedWeapons.length > 1` → 卖按钮 disabled + title 提示
- `UISystem._sellWeapon`: 安全检查失败 → `_showShopError('shop_sell_min')` toast 提示

**i18n** (`src/charsData/ui_charsData.json`):
- `shop_sell_min`: "🛡️ 至少保留 1 把武器"

## 五、状态机

```
                    ┌─────────────┐
                    │    menu     │ ← 启动
                    └──────┬──────┘
                           │ startGame
                           ▼
                    ┌─────────────┐
        ┌──────────▶│   playing   │ ◀── closeShop
        │           └──────┬──────┘
        │                  │ wave end
        │                  ▼
        │           ┌─────────────┐
        │           │    loot     │ → onAllChestsOpened
        │           └──────┬──────┘
        │                  │ (settlement check)
        │                  ├─ isFinalLevel && !endlessMode → victory
        │                  ├─ levelUpPending               → levelup
        │                  └─ else                         → shopping
        │                  ▼
        │           ┌─────────────┐         ┌─────────────┐
        │           │   levelup   │────────▶│   shopping   │
        │           └─────────────┘ close  └──────┬──────┘
        │                                         │ closeShop
        │                                         ▼
        │                                  ┌─────────────┐
        │                                  │   playing   │ (循环)
        │                                  └─────────────┘
        │
        │  player dies
        ▼
  ┌─────────────┐
  │   gameover  │ → showGameOver
  └─────────────┘

  ┌─────────────┐
  │   victory   │ → showVictory (仅 final level + 非 endless)
  └─────────────┘
```

新增状态: `victory` (类似 gameover 但不同 UI, 走"🏆 通关"路径)
