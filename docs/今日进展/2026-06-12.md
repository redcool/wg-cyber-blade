## 4. 波次系统大修（核心 Bug 修复）

**问题**：`_buildTypePool(config)` 读 `config.availableTiers`，但 CSV 输出的是 `availableBehaviors`。
结果永远 `availableTiers = undefined` → 回退 `[1]` → 只生成 basic + fast 两种敌人。

**修复**：
- 新增 `BEHAVIOR_TIER_MAP`：behavior 名 → tier 的反查表
- `_buildTypePool` 优先使用 `availableBehaviors`（CSV 数据路径），若没有则回退 `availableTiers`
- `ENEMY_TIERS` 新增 tier 4（特殊机制敌人），tier 1/3 补充 swarm/summoner

## 5. 新增 9 种敌人类型

| 敌人 | 行为 | 说明 |
|------|------|------|
| swarm | swarm | 虫群低血量群聚加速 |
| summoner | summoner | 召唤 chaser 小怪 |
| splitter | chase | 死亡分裂（splitter 机制） |
| shielded | chase | 护盾先扣盾再扣血 |
| leech | chase | 攻击回血 |
| reflect | chase | 反弹伤害给玩家 |
| freezer | chase | 攻击减速冻结 |
| frostLord | tank | Boss，冰霜领主高HP慢速 |
| shadowAssassin | blink | Boss，暗影刺客高闪避 |

所有 5 种特殊机制（splitter/shielded/leech/reflect/freezer）代码已在 `enemy.js` 实现，通过 `specialMechanic` 字段挂载。

## 6. 修复 heal/blink/mortar 行为

**之前**：`BEHAVIOR_ALIAS` 把 heal→chaser、mortar→shooter、blink→chaser，专属参数永不生效。

**现在**：
- **healer**：保持 250px 距离 → 治疗 120px 内受伤友军 → 无目标时当 chaser
- **blinker**：chaser 移动 + 每 3s 瞬移到玩家附近 + 0.15s 无敌
- **mortar**：保持 350px 距离 + 抛射迫击弹（落地范围爆炸）
- 连带修复 `bullet.js`：`isMortar` 现在从 `extra` 读取（之前永远 false）
