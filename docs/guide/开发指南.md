# CYBER BLADE · 开发指南

> 如何运行、修改、扩展游戏

---

## 一、本地运行

### 方式 1：直接打开
双击项目根目录的 `index.html` 即可运行（`file://` 协议）。
注意：Google Fonts 需要使用网络加载。

### 方式 2：HTTP 服务器
```bash
# Python 3
python -m http.server 8080 --directory /path/to/buffPrj1

# Node.js (若已安装 http-server)
npx http-server /path/to/buffPrj1 -p 8080
```
浏览器打开 `http://localhost:8080`

## 二、快速修改指南

### 2.1 添加新武器

在 `src/shop.js` 的 `allWeapons[]` 中添加：

```js
{
    id: 'myweapon',           // 唯一ID
    name: '我的武器',         // 显示名
    desc: '+20%伤害 / -10%攻速',  // 描述
    icon: '🔫',               // 图标
    slots: 2,                 // 占用槽位
    cost: 12,                 // 价格
    mods: { damageMult: 0.2, attackSpeedMult: -0.1 },
    behavior: 'bullet',       // 行为类型
    bulletCount: 1,
    bulletSpeed: 500,
    damageMult: 1.2,
    attackSpeedMult: 1.0,
    spread: 0.1,
    pierce: 0,
}
```

然后在 `src/player.js` 的 `_fireWeapon()` switch 中添加新 behavior case。
在 `src/renderer.js` 的 `drawBullet()` 中添加对应的视觉效果。
在 `src/unlock.js` 的 `weaponUnlocks[]` 中添加解锁条件。

### 2.2 添加新道具

在 `src/shop.js` 的 `allItems[]` 中添加：

```js
{
    id: 'myitem',
    name: '我的道具',
    desc: '效果描述',
    cost: 6,
    icon: '✨',
    apply: (p) => { p.someProperty += value; }
}
```

### 2.3 添加新角色

在 `src/character.js` 的 `allCharacters[]` 中添加：

```js
{
    id: 'myhero',
    name: '我的英雄',
    desc: '特点描述',
    icon: '🦸',
    unlocked: false,
    stats: {
        maxHp: 100, hpRegen: 0.5, speed: 200,
        damage: 15, attackSpeed: 1.0, attackRange: 300,
        armor: 0, dodge: 0, critChance: 0.05, critMultiplier: 2.0,
        bulletCount: 1, bulletPierce: 0, bulletSpeed: 500,
        lifeSteal: 0, pickupRange: 50, harvesting: 0, luck: 0,
    },
    weaponSlots: 6,
    unlockCondition: { type: 'totalKills', value: 200 }
}
```

### 2.4 添加新敌人类型

在 `src/enemy.js` 的 `types` 对象中添加：

```js
myenemy: {
    name: '自定义敌人',
    hp: 50,
    speed: 100,
    damage: 10,
    radius: 16,
    color: '#00ff00',
    glowColor: '#00cc00',
    xpValue: 8,
    materialValue: 3,
    attackCooldown: 1.5,
    behavior: 'chase'  // 或 'ranged'
}
```

然后在 `wave.js` 的 `_availableTypes` getter 中添加出现条件：
```js
if (lv >= 3) types.push('myenemy');
```

在 `renderer.js` 的 `drawEnemy()` 中添加形状渲染分支。

### 2.5 添加新武器行为

1. 在 `src/player.js` 中添加 `_fireMyBehavior(angle, params, target, weaponId)` 方法
2. 在 `_fireWeapon()` switch 中添加 case
3. 在 `src/bullet.js` 中添加对应的子弹更新逻辑
4. 在 `src/renderer.js` 的 `drawBullet()` 中添加视觉效果

### 2.6 添加升级选项

在 `src/stats.js` 的 `levelUpOptions[]` 中添加：

```js
{ id: 'myupgrade', name: '升级名', desc: '效果描述', icon: '⭐', apply: (s) => s.x += y }
```

## 三、平衡性调整

### 难度调节
- `wave.js`: 修改 `levelDuration`（关卡时长）、`spawnInterval`（生成间隔）
- `enemy.js`: 修改 `types` 中的敌人基础属性
- 难度缩放公式：`wave.js > _difficultyScale` getter

### 经济调节
- `shop.js`: 修改武器/道具的 `cost`
- `enemy.js`: 修改敌人的 `materialValue`（掉落金币量）
- `main.js`: 修改 `_dropMaterials()` 中的掉落数量（10%概率掉2枚，90%掉1枚）
- 定价公式：`基础价 + 当前波次 + 基础价 × 0.1 × 波次`，乘以品质倍率

### 玩家强度
- `stats.js`: 修改 `baseStats` 默认属性
- `character.js`: 修改角色基础属性
- `player.js`: 修改 `_calcDamage()` 暴击逻辑

### 音频调节
- `audio.js`: 修改 `_bgmGain.gain.value`（BGM 音量，默认 0.06，同时控制文件和程序化 BGM）
- `audio.js`: 修改 `_sfxGain.gain.value`（音效音量，默认 0.15）
- 各音效内部还有单独的 `gain` 倍率（0.4~1.0）
- BGM 曲目列表：`sounds/tracks.json` 中定义（动态加载，无需改 `audio.js` 代码）
  - 格式：`[{ "id": "combat1", "file": "sounds/bgm_1.m4a", "name": "战斗主题1" }]`
  - 添加新曲目只需放入 `sounds/` 目录并在 `tracks.json` 追加条目
- BGM 暂停/恢复：进入商店/升级/宝箱奖励时自动 `AudioSystem.pauseBGM()`，返回战斗自动 `AudioSystem.resumeBGM()`
- BGM 回退模式：`bpm = 120`，8 拍循环，可在 `_playBGMLoop()` 中调整
- 切换曲目：`AudioSystem.switchBGM('combat1')`

### 拾取吸引调节
- `main.js`: 修改 `_updateMaterials()` 中的 `attractSpeed`（金币吸引速度，默认 300px/s）和 `collectDist`（收集距离，默认 10px）
- `wave.js`: 修改 `MedkitSystem.update()` 中的 `attractSpeed`（医疗包吸引速度，默认 250px/s）
- `wave.js`: 修改 `ChestSystem.update()` 中的 `attractSpeed`（宝箱吸引速度，默认 180px/s）
- 吸引范围 = `player.pickupRange + 8/radius`，受角色基础属性和道具影响

## 四、跨局存档

存储位置：浏览器 `localStorage`，key = `cyberblade_unlocks`

```js
// localStorage 数据结构
{
    stats: {
        totalKills: number,
        totalMaterials: number,
        totalLevels: number,
        maxLevel: number,
        highestLevel: number,
        totalPlayTime: number,
    },
    unlockedWeapons: string[],
    unlockedCharacters: string[],
}
```

如果要重置解锁数据，在浏览器 DevTools Console 中运行：
```js
localStorage.removeItem('cyberblade_unlocks')
```

## 五、常见问题

### Q: 修改后页面没变化？
A: 浏览器缓存问题。请按 `Ctrl+Shift+R`（或 `Ctrl+F5`）硬刷新。
如果使用 HTTP 服务器，确保重启服务器。

### Q: 添加新文件后控制台报错 "undefined"？
A: 检查 `index.html` 中 `<script>` 的加载顺序，新文件需要放在正确的位置。

### Q: 游戏卡顿？
A: 检查粒子数量是否过多（`particle.js` 的 `emit()` 参数）。初期可降低 count 值。

### Q: 想支持触屏操作？
A: 当前只有键盘+鼠标输入。可以扩展 `input.js` 添加 touch 事件监听。

## 六、已知限制

1. 无移动端触屏支持
2. 无 PWA / Offline 支持
3. 画面大小固定为 3000×3000 世界
4. 解锁数据存 localStorage，清除浏览器数据即丢失
5. BGM 使用 Web Audio API 的 fetch + decodeAudioData 播放本地文件（`sounds/bgm_*.m4a`），不支持远程流媒体
6. 音频基于 Web Audio API 程序化生成（非预录音频），音质受浏览器实现影响
