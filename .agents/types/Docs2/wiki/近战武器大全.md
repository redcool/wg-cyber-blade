# 古今中外近战武器大全

> 系统收录全球各文明、各历史时期的近战武器，按兵器类型与功能分类，附详细结构说明、使用特点与历史背景。

## 总论：近战武器分类方法

### 一、什么是近战武器

近战武器（Melee Weapon），指在肉搏格斗中直接用于杀伤对手的非投射类兵器。与远程武器（弓弩、投掷武器等）不同，近战武器要求使用者与目标进入直接接触范围才能发挥杀伤效果。

### 二、分类体系

本手册采用**功能—形制—使用方式**三维分类法，将全球近战武器分为十大类：

| 大类编号 | 类别名称 | 涵盖范围 |
|----------|----------|----------|
| 壹 | 刀剑类（直刃切割武器） | 中国剑/刀、日本刀、欧洲长剑等 |
| 贰 | 短刃类 | 匕首、短剑、军刀等 |
| 叁 | 长柄/枪矛类 | 枪、戟、矛、戈等 |
| 肆 | 打击/钝器类 | 锤、鞭、锏、棍、杖等 |
| 伍 | 投掷类武器 | 飞刀、手里剑、标枪等 |
| 陆 | 软兵器与奇门兵器 | 九节鞭、流星锤、绳镖等 |
| 柒 | 盾甲与防御装备 | 盾牌、盔甲、护手等 |
| 捌 | 复合型/多刃武器 | 斧钺、钩镰、戟类复合兵器等 |
| 玖 | 现代近战武器 | 刺刀、格斗刀、警棍等 |
| 拾 | 特殊/礼仪用兵器 | 权杖、仪式用刀、军刀礼宾等 |

### 三、武器构造通用术语

- **刃**：兵器具有切割能力的锋利边缘
- **脊**：刀刃对面较厚的一面，增加结构强度
- **血槽**（Fuller）：刃面上的凹槽，用于减轻重量、平衡重心
- **柄**：握持部分，可缠绕绳索、皮革或装饰
- **格/护手**（Crossguard）：防止手滑至刃上的屏障
- **首**：柄末端，配重并防止滑落

### 四、兵器演进规律

1. **石器时代**：木石骨器（石斧、骨矛）
2. **青铜时代**：铜戈、铜剑、青铜短剑
3. **铁器时代**：铁剑、铁矛、铁戟
4. **冷兵器鼎盛期**（古典时代至中世纪）：各类兵器形制成熟，技术精湛
5. **火器时代**：冷兵器逐步退出一线作战，保留仪仗、指挥、特种用途
6. **现代**：刺刀、战术刀、警用格斗装备

## 壹、刀剑类（直刃切割武器）

> 刀剑类是古今中外使用最广泛、种类最丰富的近战武器大类。刀与剑的主要区别在于：剑为双刃直兵，以刺击和切割并用；刀多为单刃，可直可曲，以切割为主。

### 1.1 中国刀剑

#### 1.1.1 青铜剑（Jian / Chinese Straight Sword）

| 项目 | 说明 |
|------|------|
| 时期 | 商周至秦汉（公元前1600年—公元3世纪） |
| 刃数 | 双刃 |
| 材质 | 早期青铜，后演化为铁/钢 |
| 长度 | 约50—100 cm |
| 特点 | 中国青铜铸造技术登峰造极，越王勾践剑保存2400余年仍锋利如初，采用复合铸造与表面硫化处理 |
| 用途 | 贵族佩剑、军官指挥兵器 |
| 代表 | 越王勾践剑、秦青铜剑 |

**历史说明**：先秦时期，剑是身份与地位的象征，"君子剑"是士人必备的佩饰。秦汉之后铁剑逐渐取代青铜剑，至唐代剑进一步演化为仪仗与文学意象。

### 1.2 日本刀剑

#### 1.2.1 武士刀/日本刀（Katana）

| 项目 | 说明 |
|------|------|
| 时期 | 平安时代末期至今（约12世纪—） |
| 刃数 | 单刃 |
| 材质 | 玉钢（Tamahagane）经折叠锻打（可达1000层以上） |

| 结构 | 刃（Ha）、脊（Shinogi）、刃文（Hamon）、锤头（Kashira）、镡/护手（Tsuba） |

**锻造工艺**：顶级日本刀需经折叠锻打、覆土（Differentiated Hardening）、多次淬火等工序，总耗时可达数周至数月。一把优质武士刀的刃纹独一无二，具有极高的艺术与收藏价值。

## 贰、短刃类

> 短刃类武器长度通常不足40 cm，以刺杀和近身缠斗为主要功能，常作为副武器、暗器或警用装备。

### 2.1 中国匕首

| 武器 | 时期 | 特点 |

## 附录：东西方兵器文化对比

### 刀 vs 剑

| 比较维度 | 东方（中国/日本） | 西方（欧洲） |
|----------|------------------|-------------|
| 主流武器 | 以**刀**为主（环首刀→唐刀→弯刀） | 以**剑**为主（短剑→长剑→细剑→军刀） |
| 原因 | 骑兵冲砍需要弯刃斩击；铁产量提升使长刀成本降低 | 板甲发展需要刺击穿透；决斗文化推动长剑精细化 |
| 文化象征 | 刀：军功、武勇；"卸刀明志" | 剑：贵族身份、骑士精神；"拔剑"象征挑战 |

### 枪矛：东方 vs 西方

| 比较维度 | 中国 | 欧洲 |
|----------|------|------|
| 发展路径 | 枪法体系极为丰富（杨家枪、少林枪等） | 马其顿方阵长矛→骑士长枪 |
| 武术地位 | "百兵之王"，枪法流派繁多 | 更多作为战场阵列兵器，武术化程度不如中国 |
| 日本特色 | — | 薙刀作为女性武器有独特文化地位 |
