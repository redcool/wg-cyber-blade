# 魔法武器大全

> 系统收录古今中外神话、传说、文学作品及Roguelike游戏中的魔法武器，按元素类型分类，附详细结构说明、魔法特性与游戏化设计参考。

---

## 总论：魔法武器分类方法

### 一、什么是魔法武器

魔法武器（Magic Weapon），指超越物理法则、蕴含超自然力量的武器。与常规武器不同，魔法武器通过附魔、神圣祝福、元素共鸣、黑暗仪式等方式获得额外能力，包括但不限于：元素伤害、吸血、能量场操控、空间传送、诅咒与祝福等。

### 二、分类体系

本手册采用**元素类型—功能特性**二维分类法，将魔法武器分为九大类：

| 大类编号 | 类别名称 | 涵盖范围 |
|----------|----------|----------|
| 壹 | 火属性魔法武器 | 火焰伤害、燃烧、爆炸、高温 |
| 贰 | 水/冷冻/冰属性魔法武器 | 冰冻、冷冻、减速、水流伤害 |
| 叁 | 风/空气属性魔法武器 | 风刃、浮空、速度、漩涡 |
| 肆 | 雷电属性魔法武器 | 闪电、麻痹、静电、电磁 |
| 伍 | 能量/力场属性魔法武器 | 念动力、力场屏障、能量光束、引力操控 |
| 陆 | 黑魔法（攻击吸血/暗影） | 生命汲取、暗影伤害、诅咒、灵魂操控 |
| 柒 | 土/岩石属性魔法武器 | 岩石操控、地震、矿物附魔 |
| 捌 | 光/神圣属性魔法武器 | 圣光、净化、复活、驱魔 |
| 玖 | 复合/混沌/特殊属性 | 多重元素、空间、时间、诅咒武器 |

### 三、Roguelike武器属性通用术语

- **元素伤害**（Elemental Damage）：武器附加的非物质伤害类型
- **吸血**（Life Steal）：命中时恢复持有者一定生命值
- **附魔**（Enchantment）：赋予武器的永久魔法增益
- **诅咒**（Curse）：武器带来的负面效果
- **充能**（Charges）：魔法武器使用次数限制
- **词缀**（Prefix/Suffix）：Roguelike中随机生成的武器修饰语
- **稀有度**（Rarity）：普通 → 魔法 → 稀有 → 传奇 → 古代/神话

## 附录：Roguelike魔法武器设计通用框架

### A. 武器生成公式

```
武器名称 = [词缀前缀] + [武器类型] + [词缀后缀]
示例：烈焰之 · 长剑 · 冰霜
      雷霆 · 战锤 · 暗影
      风暴 · 法杖 · 圣光
```

### B. 属性矩阵

| 属性 | 伤害类型 | 控制效果 | 特殊效果 |
|------|----------|----------|----------|
| 火 | DoT燃烧 | 点燃 | 爆炸、自燃 |
| 冰 | 直接冰冻伤害 | 减速、冻结 | 碎冰、冰墙 |
| 风 | 切割/穿透 | 浮空、牵引 | 加速、飞行 |
| 雷 | 爆发伤害 | 麻痹 | 闪电链、静电 |
| 能量 | 持续能量伤害 | 力场压制 | 能量护盾 |
| 暗影 | 直接暗影伤害 | 恐惧 | 吸血、诅咒 |
| 土 | 物理+地裂伤害 | 震退 | 石肤、地震 |
| 光 | 对邪恶特攻 | 净化 | 治疗、复活 |

### C. 稀有度阶梯

| 稀有度 | 词缀数量 | 特殊属性 | 获取途径 |
|--------|----------|----------|----------|
| 普通（Common） | 0 | 无 | 基础掉落 |
| 魔法（Magic） | 1 | 基础元素伤害 | 宝箱/商店 |
| 稀有（Rare） | 2 | 控制效果 | 精英掉落 |
| 传奇（Legendary） | 3 | 独特机制 | Boss掉落/合成 |
| 神话（Mythic） | 4+ | 终极技能 | 隐藏Boss/多阶段合成 |

### D. 武器类型×元素矩阵（Roguelike适用）

| 武器类型 | 推荐元素 | 设计思路 |
|----------|----------|----------|
| 剑/直刃 | 多元素适配 | 平衡型，适合附魔 |
| 锤/钝器 | 雷/土 | 冲击+元素爆发 |
| 法杖 | 全元素 | 远程法术投射 |
| 弓/弩 | 火/风/冰 | 远程精准打击 |
| 匕首/短刃 | 暗影/火 | 高暴击+吸血 |
| 长枪/矛 | 雷/冰 | 穿透+控制 |
| 斧 | 火/土 | 高爆发+地面效果 |
| 权杖/ scepter | 光/能量 | 高魔伤+辅助 |
