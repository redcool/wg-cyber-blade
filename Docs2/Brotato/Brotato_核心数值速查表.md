## 伤害计算公式

### 武器最终伤害
```
最终伤害 = 基础伤害 + (属性值 × 缩放百分比)
```

**示例**:
- Knife Tier 4: 20(80%)
- 近战伤害30点时: 20 + (30 × 0.8) = 44

### 敌人成长公式
```
当前HP = 基础HP + (波次 - 1) × 每波HP增长
当前伤害 = 基础伤害 + (波次 - 1) × 每波伤害增长
```

### 护甲减伤
```
每点护甲 = 需要多6.66%伤害才能击杀
实际减伤 ≈ 护甲 / (护甲 + 15) × 100%
```

### 难度乘数
```
最终敌人强度 = 危险等级倍率 × 无障碍滑块倍率

Danger 3: 1.12倍
Danger 4: 1.26倍
Danger 5: 1.40倍
```

## 商店价格计算

### 物品最终价格
```
最终价格 = floor((基础价格 + 波次 + 基础价格 × 0.1 × 波次) × 价格系数)
最低价格: 1材料
```

**示例**:
```
SMG (基础20) 在第5波:
= floor((20 + 5 + 20×0.1×5) × 1.0)
= floor(25 + 10)
= 35材料
```

### 刷新价格
```
刷新增量 = floor(0.40 × 波次)  [最小1]
首次价格 = floor(波次 × 0.75) + 刷新增量
第N次价格 = 首次价格 + (N-1) × 刷新增量
```

**Wave 10示例**:
- 增量 = floor(0.40 × 10) = 4
- 首次 = floor(10 × 0.75) + 4 = 7 + 4 = 11
- 第二次 = 15, 第三次 = 19...

## 武器DPS计算

### DPS公式
```
DPS = 伤害 / 攻击速度
```

**注意**: 
- 不包含特殊效果
- 不包含暴击
- 近战武器需考虑范围和多目标

### 示例计算

**Tier 4 Knife**:
- 伤害: 20 + (30近战伤害 × 0.8) = 44
- 攻速: 0.78秒
- DPS: 44 / 0.78 = 56.4

**考虑暴击** (50%暴击率, 4倍暴击):
- 平均伤害 = 44 × (0.5 × 4 + 0.5 × 1) = 44 × 2.5 = 110
- 实际DPS ≈ 110 / 0.78 = 141
