五、波次生成系统（Wave System）
🧠 核心设计思想

每一波 = 一个“压力组合”

🎯 波次结构
Wave {
  Duration
  EnemyBudget
  SpawnPattern
  DifficultyScale
}
💰 EnemyBudget（关键）

每波有一个“预算”，决定生成多少敌人：

总预算 = WaveNumber × 10
🧮 生成逻辑
while budget > 0:
    enemy = 随机敌人（根据权重）
    if enemy.cost <= budget:
        spawn(enemy)
        budget -= enemy.cost
📈 难度曲线
波数	特点
1~3	教学
4~6	引入远程
7~10	混合敌人
11+	特殊敌人

随机分布
