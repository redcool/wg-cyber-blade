一、稀有度与保底机制
稀有度	出现概率	保底次数	特点
Common	60%	无	基础数值加成
Rare	25%	每3次必出一次	数值加成 + 小机制
Epic	10%	每10次必出一次	强机制 / 流派强化
Legendary	5%	每20次必出一次	改变玩法 / 核心件

说明：保底机制可用于宝箱、商店、升级卡

二、商店刷新表

每次刷新3~5件商品，按类型和稀有度分布：

类型	Common	Rare	Epic	Legendary
武器	40%	15%	4%	1%
道具	15%	10%	6%	3%
临时Buff	5%	0%	0%	0%
材料/资源	10%	0%	0%	0%

商店刷新算法：

每格随机生成稀有度
若当前未出现Rare/Epic/Legendary，根据保底规则提升概率
商品类型随机，但可根据玩家Build偏向调整（流派加成）

流派加权：
玩家已选标签 +20~30%权重出现在候选卡中

六、随机生成流程示例（伪代码）
def generate_shop(player_build_tags):
    items = []
    for i in range(random.randint(3,5)):
        rarity = roll_rarity_with_guarantee()
        item_type = roll_item_type()
        item = select_item(rarity, item_type)
        # 流派加权
        if item.tags.intersect(player_build_tags):
            increase_probability(item)
        items.append(item)
    return items

def generate_chest(player_build_tags, chest_type):
    rarity = roll_chest_rarity(chest_type)
    item_count = random.randint(1,3)
    items = []
    for _ in range(item_count):
        item_type = roll_item_type()
        item = select_item(rarity, item_type)
        if item.tags.intersect(player_build_tags):
            increase_probability(item)
        items.append(item)
    return items

def generate_levelup_cards(player_build_tags):
    candidate_cards = []
    for _ in range(3,6):
        rarity = roll_rarity_with_guarantee()
        card_type = roll_card_type(rarity)
        card = select_card(rarity, card_type)
        if card.tags.intersect(player_build_tags):
            increase_probability(card)
        candidate_cards.append(card)
    return candidate_cards
七、保底与防崩机制
每次商店刷新 / 宝箱生成 / 升级卡生成前检查：
是否满足保底次数
若未出现指定稀有度，则提升概率
流派标签增加加权，但总概率不超过 100%
限制一次只能选 1~3 张卡/道具/武器，保证策略选择紧张
